# Flashcard Quiz App

## Description
A flashcard app for studying. Users can view questions on the front, answers on the back, and navigate through cards. Users can add, edit, and delete flashcards.

## Features
- Show question/answer with a "Show Answer" button
- Navigate cards with "Next" and "Previous"
- Add, edit, delete flashcards
- Simple and clean user interface
